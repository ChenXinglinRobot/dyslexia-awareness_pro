# 字向每人 — 阅读障碍科普网站设计构思

## 三种风格方向

### 方向一：「纪录片式沉浸叙事」
以电影纪录片的视觉语言为核心，深色基调搭配高对比度文字，全屏视差滚动驱动叙事节奏，每个章节如同纪录片的一幕。强调情绪张力与信息递进的戏剧性。
- **概率**：0.04

### 方向二：「解构文字 · 认知重建」
以"文字"本身作为设计主角——汉字的笔画、偏旁、结构成为视觉元素。通过拆解、重组、模糊、清晰化等视觉手法，让访客在浏览过程中经历"文字从混乱到清晰"的认知旅程。深蓝墨色为底，辅以暖光点缀，如同黑夜中逐渐亮起的灯。
- **概率**：0.07

### 方向三：「温暖教育 · 柔光科普」
以教育类杂志的排版美学为参照，浅暖色调为主，大量留白与精致的卡片式布局，强调可读性与专业感的平衡。适合机构官网风格，但可能缺乏足够的情绪冲击力。
- **概率**：0.03

## 选定方向：「解构文字 · 认知重建」

### Design Movement
**Deconstructivist Typography** 融合 **Cinematic Scroll Narrative** — 以瑞士国际主义排版的严谨骨架为基础，叠加解构主义对文字形态的实验性处理。每个页面段落都是一次"从混沌到秩序"的微型叙事。

### Core Principles
1. **文字即主角**：汉字的笔画、偏旁、结构是全站最核心的视觉元素，而非装饰性插图
2. **渐进式清晰**：视觉从模糊/碎片化逐步走向清晰/完整，映射阅读障碍者的认知挣扎与希望
3. **深色沉浸**：深墨蓝底色营造专注氛围，让文字本身成为光源
4. **克制的温度**：在冷峻的视觉框架中，通过暖色光点（琥珀/金色）传递人文关怀

### Color Philosophy
- **主底色**：深墨蓝 `oklch(0.15 0.02 250)` — 代表阅读障碍者面对文字时的"深水区"感受
- **品牌主色**：琥珀暖光 `oklch(0.72 0.14 70)` — 如同黑暗中的灯火，代表理解与希望
- **辅助色**：柔白 `oklch(0.92 0.01 90)` — 清晰的文字，代表"被看见"
- **强调色**：珊瑚红 `oklch(0.65 0.18 25)` — 用于警示、数据高亮、关键信息
- **过渡色**：雾蓝 `oklch(0.45 0.04 240)` — 中间态，模糊与清晰之间

### Layout Paradigm
**垂直叙事流 + 全宽沉浸段**：单页锚点架构，五个一级导航对应五个主要叙事段落。每段使用全屏宽度，通过视差、渐变过渡和动态元素创造"翻页"感。拒绝居中对称的卡片网格——改用不对称的文字布局、大面积留白与突破性的字号对比。

### Signature Elements
1. **笔画粒子系统**：汉字偏旁部首作为装饰性粒子，在背景中缓慢漂浮，暗示文字的碎片化体验
2. **渐进清晰遮罩**：关键文字从模糊到清晰的 CSS 动画过渡，让访客"体验"阅读障碍者的视觉世界
3. **光线引导**：琥珀色光晕从画面某处射出，引导视线流动，象征"被理解的温暖"

### Interaction Philosophy
交互即叙事——每一次用户操作（滚动、点击、拖动滑块）都在推进"从困惑到理解"的故事线。模拟体验让用户"成为"阅读障碍者；排版调适让用户"成为"改变者。交互不是功能，是共情的载体。

### Animation
- **入场动画**：文字从碎片状态（偏旁分离、笔画散落）聚合为完整汉字，时长 600-800ms，ease-out
- **滚动触发**：基于 Intersection Observer 的渐进式揭示，每个模块在进入视口时"点亮"
- **微交互**：按钮按下时 scale(0.97)，悬停时琥珀色光晕扩散
- **模拟体验动画**：文字抖动、模糊、重叠——强度可控，不致眩晕
- **过渡**：段落间使用渐变色带 + 微妙的 translateY 位移，营造"深潜"感

### Typography System
- **中文标题**：Noto Serif SC, 700 weight — 庄重、有力、学术感
- **中文正文**：Noto Sans SC, 300/400 weight — 清晰、现代、易读
- **英文装饰/数据**：Space Grotesk — 几何感强、现代、与汉字的方块结构呼应
- **字号层级**：首屏标题 clamp(3rem, 8vw, 6rem)；章节标题 clamp(2rem, 4vw, 3.5rem)；正文 1rem/1.125rem

### Brand Essence
**字向每人** — 一个让普通人真正理解阅读障碍的沉浸式科普体验，为关心教育公平的家长、教师和公众而生，通过"亲历"而非"告知"建立认知共情。
- 三个性格形容词：**沉浸的、温暖的、有力的**

### Brand Voice
标题与行动号召的语气：克制而有力，不煽情但有温度，像一位冷静的纪录片旁白。
- 示例标题：「一道乘法题，归零了整个世界」
- 示例 CTA：「亲手调一调，感受文字的呼吸」

禁止出现的表达：「欢迎来到我们的网站」「立即开始」「点击了解更多」等通用空话。

### Wordmark & Logo
品牌标识概念：将"字"字的笔画进行解构——宝盖头保持完整（象征保护），下方笔画以粒子形态散开又重聚（象征从碎片到完整的旅程）。纯图形标识，无文字，琥珀色为主。

### Signature Brand Color
**琥珀暖光** `oklch(0.72 0.14 70)` — 这是属于"字向每人"的颜色：在深色背景中，它是唯一的暖色光源，象征理解、希望与陪伴。
